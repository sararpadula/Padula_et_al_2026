# Padula_et_al_2026

dsData.csv contains the raw breeding data for the Boulder Chickadee Study, which includes all metadata for each black-capped and mountain chickadee included in the analysis.

population_SMI.csv contains a row for each individual included in the population-level analysis.

paired_SMI.csv contains a row for each breeding pair included in the analysis.

Padula_2025_SSD_analysis.Rmd is an Rmarkdown file that separately tests for sexual size dimorphism using linear model within all individuals, as well as just within breeding pairs. It also includes the permutation analyses.

Pdaula_2025_wing_diff_analysis.Rmd is an Rmarkdown file that examines the relationship between wing length difference and all included parental quality and reproductive variables. It also tests female wing length predicts male wing length within breeding pairs.
